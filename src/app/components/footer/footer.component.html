Desarrollado con Tecnologías Modernas
Este proyecto fue desarrollado utilizando Angular v19 con PrimeNg, PrimeFlex y Tailwindcss para la interfaz de usuario, para tarabajar en entorno local se utilizo archivos de configuración de Docker como Dockerfile y docker-compose, tienen su respectiva modificación para desplegarse en producción con el archivo de configuracion de Nginx, pero actualmente se aloja en Netlify ya que el servicio es gratis.
